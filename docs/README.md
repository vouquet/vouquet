vouquet
===

* vouquet (Virtual coin bOUQUET) は、個々の仮想通貨の思想を元に仮想通貨という種子を育て、立派な花束(資産)を作成するプロジェクトです。
* 取引ロジックは非公開ですが、APIの利用周り等は公開していきます。
